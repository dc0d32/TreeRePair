Detailed Description

Implements a XML tree printer, i.e., a given tree is transformed into a XML document.

bool XmlTreePrinter::convert [private]

If true, the tree to print (which is then supposed to be a binary representation of a unranked tree) is on-the-fly converted to the original unranked tree (i.e., the unranked tree is written to the output file).
