void TreeConstructor::shareTree ( TreeNode *  pNode  )  [protected]

This method checks if the given subtree is shareable and if so it checks if there was already an equal subtree encountered (in this case the subtree is replaced by a nonterminal of the DAG grammar). If it is shareable and no equal subtree was encountered until now, it is added to the subtrees hashmap in order to be able to share it at a later date.

Parameters:
pNode the root of the subtree which shall be shared
