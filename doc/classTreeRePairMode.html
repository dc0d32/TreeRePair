Detailed Description

An instance of this (singleton) class reflects the mode in which a tool of the TreeRePair distribution is executed.

Member Enumeration Documentation

anonymous enum
Enumerator:
COMPRESS 

Indicates that we are executing the TreeRepair tool (compressor).

DECOMPRESS 

Indicates that we are executing the TreeDePair tool (decompressor).

TOOL_DATAFILE 

Indicates that we are executing the TreeRePairTool in order to generate a data file which serves as a input for gnuplot (the class DataFileCreator is used).

TOOL_STRUCTURE 

Indicates that we are executing the TreeRePairTool in order to write out the XML document tree of an input XML document (the class StructureRetriever is used).

TOOL_SUMMARY 

Indicates that we are executing the TreeRePairTool in order to write out a latex file summarizing multiple runs of different compressors (the class SummaryCreator is used).

TOOL_BATCH_MIN_DAG 

Indicates that we are executing the TreeRePairTool in order to obtain informations about the minimal DAG of multiple input XML documents.

TOOL_BATCH_TREEREPAIR 

Indicates that we are executing the TreeRePairTool in order to run the TreeRepair tool on multiple input XML documents in a batch.

TOOL_BATCH_BPLEX 

Indicates that we are executing the TreeRePairTool in order to run the BPLEX tool on multiple input XML documents in a batch.

TOOL_BATCH_REPAIR 

Indicates that we are executing the TreeRePairTool in order to run the Extended-Repair tool on multiple input XML documents in a batch.

TOOL_BATCH_GZIP 

Indicates that we are executing the TreeRePairTool in order to run the gzip tool on multiple input XML documents in a batch.

TOOL_BATCH_BZIP 

Indicates that we are executing the TreeRePairTool in order to run the bzip2 tool on multiple input XML documents in a batch.

TOOL_BATCH_XMILL 

Indicates that we are executing the TreeRePairTool in order to run the XMill tool on multiple input XML documents in a batch.

TOOL_CREATE_DATABASE 

Indicates that we are executing the TreeRePairTool in order to create an empty database file.

TOOL_MEMORY_USAGE 

Indicates that we are executing the TreeRePairTool in order to measure the memory usage of another tool.

TOOL_BPLEX_GRAMMAR_ANALYZER 

Indicates that we are executing the TreeRePairTool in order to obtain information about a grammar given by a BPLEX-like formatted file.

TOOL_DATA_COLLECTOR 

Indicates that we are executing the TreeRePairTool in order to collect the character data and attribute values conatined in an XML document.

TOOL_ANALYZE 

Indicates that we are executing the TreeRePairTool in order to obtain informations about a given XML document.

TOOL_MINIMAL_DAG_INFO 

Indicates that we are executing the TreeRePairTool in order to obtain informations about the minimal DAG of an input XML document tree.

OPTIMIZE_FILE_SIZE 

Indicates that we are executing the TreeRePair tool in file size optimization mode.

OPTIMIZE_EDGES 

Indicates that we are executing the TreeRePair tool in number of edges optimization mode.

OPTIMIZE_DISABLE 

Indicates that that the pruning step shall be skipped.
