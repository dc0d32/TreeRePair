Detailed Description

This class implements a production of a linear SLCF tree grammar. It keeps track of its reference count (how many nodes are labeled by the left-hand side of this production) and of its transitive reference count (how often does the right-hand side of this production occur in the tree which is represented by the corresponding grammar). In addition it maintains a hashed set of all nodes labeled by its left-hand side.
