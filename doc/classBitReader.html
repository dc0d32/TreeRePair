string BitReader::readBitString ( unsigned int  length,
bool  lookAhead = false 
)

This methods returns a certain number of bits from the input stream and optionally reads (i.e., deletes) them from the input stream.

Parameters:
length The number of bits to read. If the end of the input stream is reached, a smaller number of bits is returned.
lookAhead This optional parameter specifies if the bits read from the input stream shall be absorbed. The default is to not absorbed them, i.e., the next call to this method will return them again.
Returns:
the bits read are returned as a string
