Detailed Description

This class implements a symbol which can either be a terminal symbol or a nonterminal.

Constructor & Destructor Documentation

Symbol::Symbol ( bool  leftChild = false,
bool  rightChild = false 
)

The constructor of the class. The parameters of it are used when creating terminal symbols in the binary model. If a terminal symbol exhibits only one child then, with their help, it can be easily determined if it is a left or right child.

Parameters:
leftChild specifies if the symbol has a left child (default is false)
